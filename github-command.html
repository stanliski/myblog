GitHub上传代码

在需要上传的项目目录下，创建一个.git文件夹

$git init

把当前路径下的所有文件添加到待上传的文件列表

$git add .

或者

$git add filename
$git commit –m “xxx”
$git remote add origin git@github.com:stanliski/demo.git

把代码上传到服务器上


$git push –u origin master

把代码拉回本地

$git pull origin master
